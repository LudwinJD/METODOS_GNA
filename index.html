MÉTODOS DE GENERACIÓN DE NÚMEROS ALAETORIOS


Método Del Cuadrado Medio

Comienza con un número inicial (semilla). Este número es elevado al cuadrado. Se escogen los dígitos del medio de este nuevo número (según los dígitos que se deseen) y se colocan después del punto decimal. Este número conforma el primer número random.

La fórmula del método del cuadrado medio es:
Xn+1 = (Xn2) / 10d

Congruencia Lineal

La congruencia lineal es un método popular para la generación de números aleatorios. Utiliza una relación de recurrencia lineal para producir una secuencia de números, donde cada número depende del anterior a través de una fórmula específica de congruencia.

La fórmula de congruencia lineal para la generación de números aleatorios es:
Xn+1 = (a * Xn + c) mod m

Congruencia Mixta

El método congruencial mixto es otra variante del método congruencial utilizado para generar secuencias pseudoaleatorias.

La fórmula de congruencia mixta para la generación de números aleatorios es:
Xn+1 = (a * Xn + c) mod m

Método Multiplicativo

El método congruencial multiplicativo es un algoritmo utilizado para generar secuencias pseudoaleatorias. Este tipo de secuencias son determinísticas y se generan a partir de una fórmula matemática, pero su comportamiento se asemeja al de una secuencia de números aleatorios verdaderos.

La fórmula del método multiplicativo para la generación de números aleatorios es:
Xn+1 = (a * Xn) mod m
